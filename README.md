# Embrace Jewelry

This repository contains the source code for the **Embrace Jewelry** website.

## Getting Started

1. Clone the repo and install dependencies:
   ```sh
   git clone <YOUR_GIT_URL>
   cd embrace
   npm install
   ```
2. Run the development server:
   ```sh
   npm run dev
   ```
3. Open http://localhost:8080/ in your browser to view the site.

## Project Overview

- **Framework:** Vite with React and TypeScript
- **UI:** shadcn-ui components styled with Tailwind CSS

## Deployment

Build the production bundle with:
```sh
npm run build
```
and serve it using any static host or platform of your choice.

## Custom Domain & Hosting

You can deploy the generated dist folder to services like Netlify, Vercel, GitHub Pages, etc., and configure a custom domain through their settings.
